💰 Smart Spend

Smart Spend helps users figure out where their money mysteriously disappears each month while efficiently tracking income, expenses, and savings 💸

🚀 Features
📊 Track income and expenses with ease
🧾 Categorize spending (food, bills, etc.)
📅 View spending over time
💡 Get insights into your financial habits
🎯 Set budget goals and try to stick to them

🛠️ Tech Stack
* Frontend: Nextjs (TypeScript), TailwindCSS
* Backend: Node.js + Express
* Database: PostgreSQL
